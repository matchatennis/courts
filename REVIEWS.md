# Reviews

User-submitted reviews of tennis courts. These are surfaced in the Matcha
Tennis apps.

## How to add a review

This file is **append-only**: add your review to the bottom, never edit or
delete an existing entry. Each review is one section in the exact shape below.

```
### <place name> — <YYYY-MM-DD>
- **court:** <MRN of the specific court, or `—` for the whole venue>

<free-form review text — one paragraph>
```

- `place name` and `court` MRN come from [`INDEX.md`](INDEX.md) / `places/`.
- Keep the heading and field order identical so the apps can parse it.

## Example

```
### Alice Marble Tennis Courts — 2026-05-17
- **court:** —

Stunning views over the bay and the nets are kept taut. Courts get windy in
the afternoon, so book a morning slot if you can.
```
